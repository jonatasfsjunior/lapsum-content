# A Estrutura atual contém:
- _products
- _data
- _img

Veja um exemplo do site no ar: https://lapsumcriativo.vercel.app/products/exemplo/
## Adicionando novos produtos
Quando tiver algo para vender, crie um .md em `_products`.
Confira `_products/exemplo.md` para entender a estruturação do arquivo.

Os Produtos devem ter um código de produto, utilizado internamente para os comentários e demais informações.
Pode ser algum aleatório, desde que seja persistente por produto.
### Imagens
Tenha um banner, 1.91:1, recomendável 1920×1005. Não excedendo 2.5mb. Veja um exemplo em `_img/banner/exemplo/`.
Um logo, 1:1, recomendável 1000x1000, fundo transparente, objeto centralizado. Veja um exemplo em `_img/logo/exemplo/`.
Renders em 16:9, recomendável 1920x1080. Exemplos em `img/galeria/exemplo/`.

Para melhor organização, cada imagem deve ter sua própria pasta com o nome referente ao produto á venda na loja.
